# Orange Mali PWA — Chunk 1: Setup & Infrastructure

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [x]`) syntax for tracking.

**Goal:** Remplacer Google Forms par une PWA mobile-first permettant aux employés de terrain de soumettre des remontées en masse avec gestion hiérarchique des rôles.

**Architecture:** React SPA avec Supabase comme backend (Auth + PostgreSQL + Row Level Security). La saisie en masse se fait via un composant tableau (une ligne = une remontée). Les formulaires sont configurables dynamiquement par le chef sans toucher au code.

**Tech Stack:** React 19 + TypeScript + Vite, Tailwind CSS v4, shadcn/ui, Supabase (Auth + DB + RLS), Vercel, Vite PWA Plugin

## Task 1: Initialiser le projet

**Files:**
- Create: `package.json`, `vite.config.ts`, `tsconfig.json`, `index.html`
- Create: `src/main.tsx`, `src/App.tsx`

- [x] **Step 1: Créer le projet Vite**

```bash
pnpm create vite@latest orange-mali-app --template react-ts
cd orange-mali-app
pnpm install
```

- [x] **Step 2: Installer les dépendances**

```bash
pnpm add @supabase/supabase-js @supabase/auth-helpers-react
pnpm add react-router-dom
pnpm add @tanstack/react-query
pnpm add dexie dexie-react-hooks
pnpm add -D vite-plugin-pwa tailwindcss @tailwindcss/vite
```

- [x] **Step 3: Configurer Tailwind v4**

Dans `vite.config.ts` :
```ts
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'
import { VitePWA } from 'vite-plugin-pwa'

export default defineConfig({
  plugins: [
    react(),
    tailwindcss(),
    VitePWA({
      registerType: 'autoUpdate',
      manifest: {
        name: 'Orange Mali Remontées',
        short_name: 'OrangeMali',
        theme_color: '#FF6600',
        background_color: '#ffffff',
        display: 'standalone',
        icons: [{ src: '/icon-192.png', sizes: '192x192', type: 'image/png' }]
      }
    })
  ],
  resolve: { alias: { '@': '/src' } }
})
```

Dans `src/index.css` :
```css
@import "tailwindcss";
```

- [x] **Step 4: Initialiser shadcn/ui**

```bash
pnpm dlx shadcn@latest init
```
Choisir : TypeScript, style Default, base color Neutral, CSS variables yes.

- [x] **Step 5: Vérifier que l'app démarre**

```bash
pnpm dev
```
Expected: app visible sur http://localhost:5173

- [x] **Step 6: Commit**

```bash
git init && git add -A
git commit -m "chore: init project with vite react ts tailwind shadcn"
```

```env
VITE_SUPABASE_PUBLISHABLE_DEFAULT_KEY=sb_publishable_94pynZjz19VWg-P5eQKI9w_iLW5drwr
```

Ajouter `.env.local` au `.gitignore`.

- [x] **Step 2: Créer `src/lib/supabase.ts`**

```ts
import { createClient } from '@supabase/supabase-js'
import type { Database } from './database.types'

export const supabase = createClient<Database>(
  import.meta.env.VITE_SUPABASE_URL,
  import.meta.env.VITE_SUPABASE_PUBLISHABLE_DEFAULT_KEY
)
```
